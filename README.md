# 技能竞赛训练日志管理系统

一个完整的全栈训练日志管理系统，支持管理员、教师、学生三种角色，响应式设计适配手机、平板、电脑。

## 功能特性

### 管理员端
- 项目管理：添加、编辑、删除竞赛项目（云计算、移动应用开发、区块链、网络安全等）
- 教师管理：创建教师账号，分配关联项目
- 学生管理：创建学生账号，分配到项目

### 教师端
- 查看所属项目的所有学生训练日志
- 按日期、学生筛选日志
- 批改评价学生日志（评分+评语）
- 查看学生列表

### 学生端
- 提交训练日志（包含训练日期、教练、课题、内容、问题、解决办法、小结、自评）
- 查看历史日志
- 查看教师评价

## 技术栈

- **后端**: Node.js + Express + MySQL
- **前端**: Vue 3 + Vite + Element Plus
- **认证**: JWT
- **响应式**: 移动端优先设计

## 快速开始

### 1. 安装依赖

```bash
# 安装后端依赖
npm install

# 安装前端依赖
cd client && npm install
```

### 2. 配置环境变量

项目根目录的 `.env` 文件已配置好数据库连接信息。

### 3. 初始化数据库

```bash
node server/config/initDb.js
```

这将创建所有必要的数据表，并创建默认管理员账号：
- 用户名: `admin`
- 密码: `admin123`

### 4. 启动项目

```bash
# 同时启动前后端（推荐）
npm run dev

# 或分别启动
npm run server  # 后端运行在 http://localhost:3000
npm run client  # 前端运行在 http://localhost:5173
```

### 5. 访问系统

打开浏览器访问: `http://localhost:5173`

## 默认账号

### 管理员
- 用户名: `admin`
- 密码: `admin123`
- 角色: 管理员

## 数据库结构

- `admins` - 管理员表
- `projects` - 项目表
- `teachers` - 教师表
- `students` - 学生表
- `training_logs` - 训练日志表

## 训练日志格式

- 训练日期
- 当日教练
- 训练课题
- 训练内容
- 存在问题
- 解决办法
- 训练小结
- 自我评价（优/良/一般/差）
- 教练评价（优/良/一般/差）
- 教练评语

## 响应式设计

系统采用响应式设计，自动适配：
- 手机（< 768px）
- 平板（768px - 1024px）
- 电脑（> 1024px）

## 后续扩展

系统架构支持后续开发微信小程序，API接口已经完全分离，可直接对接小程序端。

## 项目结构

```
.
├── server/                 # 后端代码
│   ├── config/            # 配置文件
│   ├── middleware/        # 中间件
│   └── routes/            # 路由
├── client/                # 前端代码
│   └── src/
│       ├── views/         # 页面组件
│       ├── stores/        # 状态管理
│       ├── router/        # 路由配置
│       └── utils/         # 工具函数
└── .env                   # 环境变量
```

## 开发说明

- 后端API端口: 3000
- 前端开发端口: 5173
- 前端已配置代理，开发时自动转发API请求到后端

## 注意事项

1. 确保MySQL数据库服务正常运行
2. 首次运行需要执行数据库初始化脚本
3. 生产环境请修改 `.env` 中的 `JWT_SECRET`
4. 建议使用 Node.js 16+ 版本
